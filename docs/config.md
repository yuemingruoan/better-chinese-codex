## 连接 MCP 服务器

Codex 可以连接配置在 `~/.codex/config.toml` 中的 MCP 服务器。最新 MCP 配置项请参考配置参考文档：

- https://developers.openai.com/codex/config-reference

## 通知

Codex 可以在代理完成一个回合后运行通知钩子。最新通知设置请参考配置参考文档：

- https://developers.openai.com/codex/config-reference
